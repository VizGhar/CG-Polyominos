The Goal

Place all polyominos to form predefined 2D shape

  Rules

You will receive shape you need to fill with polyominos. You can rotate polyomino by 0, 90, 180 or 270 degrees, and/or you can flip it horizontally/vertically. So there are at most 8 possible configurations for each tile.

Here is overview of every single tile used in this puzzle.
All available shapes - for example polyomino with id A forms L-like shape

  Example

If you take first test case as an example, you will need to use A and D polyominos (in this order - see Game Input section) to fill this shape:

OOO
OOO
O.O

You can fill the shape by placing tiles either like this:

DAA
DDA
D.A

Or like this:

AAD
ADD
A.D


So if you want to place tiles as shown in second example, your output should look like this:

OO;O.;O.
0 0
.O;OO;.O
0 1

So in first 2 lines, you are placing polyomino A (L-shaped) to position 0, 0 rotated such that short side of L shape is facing top of board and long side is facing left. In next 2 lines you are placing correctly rotated tile D to position 0, 1 (row, column)

  Game Input

Input

Every turn you will receive current configuration in following format:

Line 1: ids of remaining polyominos to place on board for example: ABEFN

Line 2: id of polyomino to place in this turn for example: F

Line 3: space separated h - height and w - width of game board

Following lines: h strings of length w representing game board, where

  • . - have to remain unoccupied
  • O - is not yet occupied
  • A through N - is occupied by shape

Output

Line 1: shape of polyomino as semicolon separated lines. For example line in format ..O;OOO;..O represents T shaped tile rotated by 90 degrees clockwise

Line 2: space separated row and col where the top-left corner of polyomino should be placed (top-left corner of its enclosing rectangle)
