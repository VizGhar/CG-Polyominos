gameloop
read remaining:string(15)
read current:string(1)
read h:int w:int
loop h read line:string(w)
write OO;.O;.O
0 1

STATEMENT
Place all polyominos to form predefined 2D shape

INPUT
remaining: IDs of remaining tiles
current: ID of current tile to place on board
line: single line of board - place your tiles only on 'O' symbols
